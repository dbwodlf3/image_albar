# 문제 상황
사진은 다 짤려있음.

문제가 되는건 전시장에 배치를 했는데, 어느면의 어느칸 어느 줄에 위치해 있는지 모른다는 것.

이것을 사진을 찍엇기 때문에, 찍은 사진을 바탕으로 면, 칸, 줄 을 찾아야함.

## 해결 방법

사진속의 그림에서 칸과 줄을 찾는다.

해당 그림의 성명, 소속, 나이를 기반으로 등록 사이트에 검색한다.

### 사람 손으로 수기 입력

자료 데이터가 6만5천건이다...

### 프로그램으로 등록

우선 사진의 이미지를 OCR을 할 수 있어야 한다.

이미지를 OCR한 다음에, DB 쿼리를 보내야 한다. (DB API를 제공 해 줘야함.)
